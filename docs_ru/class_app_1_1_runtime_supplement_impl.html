Подробное описание

Отвечает за инициализацию runtime подсистем. Runtime отвечает за организацию работы глобальных компонент, сети, асинхронных таймеров. RuntimeState работает в выделенном потоке, за создание которого отвечает логика приложения.

RuntimeSupplementImpl:

  • создаёт поток для runtime системы;
  • создаёт глобальный общий планировщик (для пула задач);
  • к текущему потоку привязывает планировщика для выполнения задач в очереди/потоке приложения.
